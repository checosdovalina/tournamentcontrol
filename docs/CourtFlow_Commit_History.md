## MÓDULOS DESARROLLADOS

### 1. Sistema Base
- Autenticación y sesiones
- Roles (superadmin, admin, scorekeeper, display)
- Multi-tenancy por torneo
- WebSocket para tiempo real

### 2. Gestión de Torneos
- CRUD de torneos
- Categorías y clasificaciones
- Configuración de timezone
- Logos y branding

### 3. Gestión de Jugadores
- Registro de jugadores
- Formación de parejas
- Check-in de asistencia
- Historial de partidos

### 4. Programación de Partidos
- Calendario diario
- Importación desde Excel/PDF
- Horarios configurables
- Asignación de canchas

### 5. Sistema de Canchas
- CRUD de canchas
- Asignación automática (FIFO)
- Pre-asignación
- Detección de conflictos
- URLs de streaming

### 6. Captura de Score
- Score en vivo por scorekeeper
- Captura por QR (guest)
- Historial con undo
- Auto-finalización
- Bloqueo de partidos

### 7. Timeout Processor
- Detección de ausencias (15 min)
- Soporte de timezone
- Protección retroactiva
- Sistema DQF controlado

### 8. Ready Queue System
- Cola FIFO inteligente
- Ordenamiento por plannedTime + readySince
- Preservación de posición
- Actualizaciones en tiempo real

### 9. Displays Públicos
- Display rotativo
- Display clásico filtrado
- Display de control 50/50
- Display de streaming
- Wake Lock API
- Responsive design

### 10. Sistema de Anuncios
- Upload de imágenes/videos
- Rotación automática
- Duración configurable
- Display en cambios de lado
- Banners de sponsors

### 11. Exportación/Importación
- Excel import/export
- PDF parsing
- Generación de QR
- Reportes
